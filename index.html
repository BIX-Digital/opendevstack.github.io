OpenDevStack Use Cases

Project Setup

  • Shortens ramp-up times for new projects to under 5 min
  • Sets up Wiki space, Jira and Bitbucket projects
  • Index mistakes in the production database
  • Less complex syntax
  • Primary key errors
  • Tracking changes

TEASER

OpenDevStack allows to set up required infrastructure and continuous delivery processes for new projects in less than 5 min.
The first release (Expected: Juliy, 2018) will support the Atlassian toolstack (Confluence, Jira, Bitbucket) and OpenShift
